The latest version of [commander](https://www.npmjs.com/package/commander) requires that each option short flag can only consist of a single character.

Changes:

| Old flag | New flag |
| -------- | -------- |
| `-otv`   | `--otv`  |
| `-ntv`   | `--ntv`  |
| `-otb`   | `--otb`  |
| `-ntb`   | `--ntb`  |
| `-tn`    | `-n`     |
| `-gak`   | `-g`     |
